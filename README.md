## Project
A simple CRUD API REST in NodeJS simulating a PetShop system.

## Technologies

- [NodeJS](https://nodejs.org/)
- [ExpressJS](https://expressjs.com/)
- [Sequelize](https://sequelize.org/)
- [MySQL](https://www.mysql.com/)

## Scripts
Install dependences
```sh
yarn install
```
Usage
```sh
yarn start
```
